# Alpha Dragon ÔÇö CMB / Mantle Resonance Field Lab

An interactive analog field lab for exploring CMB-style shell structure, perturbation resonance, red/blue shift analogs, and exterior vs interior horizon views.

## What this is

This project is a real-time particle-field explorer built to help visualize how structure can emerge in a dynamic radial system.

It combines:

- radial density behavior
- shell perturbation mapping
- resonance tracing
- redshift / blueshift-style flow views
- exterior vs interior horizon projection modes

This is **not** a literal cosmology solver. It is an **interactive analog field instrument** designed to make field behavior easier to see, test, and discuss.

## Core ideas explored

- **Radial density**
  - how structure accumulates by radius

- **Shell behavior**
  - how a projected outer shell can be read as a 2D boundary

- **Perturbations**
  - how small field differences appear across angle

- **Resonance**
  - how angular shell variations can be read as a live trace

- **Perspective**
  - how the same system feels different in:
    - `EXTERIOR` view
    - `INTERIOR HORIZON` view

## Main features

- Interactive particle field
- Adjustable gravity, noise, trail, spawn rate, max particles, and shell radius
- Multiple color modes:
  - `FLOW`
  - `SPLIT`
  - `MAG`
  - `TEMP`
- `EXTERIOR` and `INTERIOR HORIZON` view modes
- Right-side analysis stack:
  - Radial Density
  - Shell Heat Strip
  - CMB Resonance Trace
- HUD behavior readers:
  - shell uniformity
  - shell variance
  - resonance amplitude
  - hottest / coldest angle
  - density band counts

## Controls

### Modes
- `1` Dual Core toggle
- `2` Pair Spawn toggle
- `3` CMB Shell toggle
- `4` Density Graph toggle
- `5` Show Centers toggle
- `6` Resonance Trace toggle
- `7` Heat Strip toggle
- `O` Freeze Snapshot
- `I` View Mode toggle (`EXTERIOR` / `INTERIOR HORIZON`)
- `M` Cycle Color Mode

### Tuning
- `Q / A` Gravity up / down
- `W / S` Noise up / down
- `E / D` Trail up / down
- `R / F` Spawn-per-frame up / down
- `T / G` Max particles up / down
- `Y / H` Shell radius up / down

### Presets
- `Z` Balanced
- `X` Expansion
- `V` Perturbed
- `B` Dual Core
- `N` Pair Symmetry
- `C` Reset

## Suggested ways to explore

Start with:

- `TEMP + EXTERIOR`
- `TEMP + INTERIOR HORIZON`
- `FLOW + EXTERIOR`

Then compare:

- low noise vs high noise
- low gravity vs high gravity
- balanced vs perturbed presets

Watch how the three analysis panels change together:

- **Radial Density** = where structure is stacking
- **Shell Heat Strip** = unwrapped shell behavior
- **CMB Resonance Trace** = angular signal behavior

## Why this build exists

This project was built as a way to make shell structure, anisotropy, resonance, and observer perspective more intuitive.

The goal is not to overclaim.  
The goal is to build something interactive enough that people can see the behavior for themselves.
